# ☕ Cafetière - TD C++

Petit projet réalisé dans le cadre d'un TD de programmation orientée objet en C++.

## 📋 Description

Ce projet implémente une classe `Caffetiere` représentant une cafetière simple.

La classe contient :

- Une couleur (`couleur`)
- Un volume d'eau (`volumeEau`)
- Une méthode permettant de préparer un café (`faireCafe()`)

L'objectif est de mettre en pratique :

- Les classes et objets
- Les attributs privés
- Les constructeurs et destructeurs
- Les méthodes membres
- L'organisation du code avec fichiers `.h` et `.cpp`

## 🛠️ Technologies

- C++
- Bibliothèque standard (`<string>`, `<iostream>`)
